# Java framework for mobile apps with [TeaVM](http://teavm.org/)

## Build

#### Prerequisites

  * Java (_version >= 13_)
  * Maven
  * GIT

#### Build platform independant part

  * `git clone https://github.com/fb71/areca.git`
  * `git clone -b areca https://github.com/Polymap4/polymap4-model.git`
  * `cd areca`
  * `mvn package`
